# mayak-digit: Цифровой словарь Маяковского

![Python](https://img.shields.io/badge/python-3.8+-blue.svg)
![Status](https://img.shields.io/badge/status-active-success.svg)

**MayakDigit (Mayak-2D)** — это цифровой проект по созданию полноценного кроссжанрового авторского словаря В. В. Маяковского, совмещающий в себе инстурменты цифровой филологии и NLP.

В данный момент проект существует в статусе прототипа.

---

## 🚀 Основные функции

* **Индекс Маяка (Proximity Index):** Глобальный расчет весов близости слов по всему корпусу.
* **Семантический компаратив (Navec):** Сравнение авторского словоупотребления с общеязыковой нормой. Система находит синонимы в модели Navec и разделяет их на:
    * *Используемые автором* (с расчетом их личного контекстного веса).
    * *Проигнорированные автором* (семантические лакуны).
* **Умная лемматизация и фильтрация:** Предобработка текстов с использованием морфологического анализа для отсечения дублей и нормализации выдачи синонимов.

## Методология: Как считается "Вес"?

Для каждого вхождения искомого слова (таргета) система сканирует всё произведение целиком. В отличие от стандартных методов окон (window size), наш алгоритм учитывает глобальный контекст. 

Итоговый вес слова-соседа вычисляется по формуле затухания:

$$Weight = (decay_{dist}^{d}) \times (decay_{brk}^{n\_brks}) \times (decay_{sent}^{n\_sents})$$

Где:
* **$d$** — физическое расстояние в словах (чем дальше, тем меньше связь).
* **$n\_brks$** — количество переносов строк («лесенка»). Каждый разрыв уменьшает семантическую близость.
* **$n\_sents$** — количество границ предложений. Переход в другое предложение считается сильным барьером.

Модель автоматически суммирует веса от всех вхождений слова в тексте, что позволяет адекватно оценивать рефрены и кольцевые композиции, характерные для поэтики Маяковского.

## 📝 Планы по развитию
